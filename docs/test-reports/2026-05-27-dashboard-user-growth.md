# 测试用例 - 用户增长趋势数据看板

> 创建时间: 2026-05-27
> 需求来源: docs/requirements/2026-05-27-dashboard-user-growth.md

## 测试范围
- 功能模块: Dashboard 数据看板
- 页面: /dashboard
- 核心路径: 查看数据卡片 + 折线图

## 测试环境
- URL: http://localhost:8080
- 浏览器: Chrome (Playwright MCP)
- 需要登录: 是（Mock 账号 admin/admin）

## 用例列表

### TC-001: 数据卡片数值与 Mock 数据一致
- **优先级**: P0
- **前置条件**: 用户已登录，成功进入 /dashboard 页面
- **测试步骤**:
  1. 登录系统并进入 /dashboard 页面
  2. 等待页面数据加载完成
  3. 截图后对比三个卡片标题（今日新增用户、总用户数、7日平均日增）和数值显示
- **预期结果**:
  - "今日新增用户"卡片显示整数，如 `128`
  - "今日新增用户"卡片辅助指标显示日环比增长率，如 `+12.5%`
  - "总用户数"卡片显示累计用户总数，如 `24,580`
  - "总用户数"卡片辅助指标显示较昨日净增数，如 `较昨日 +128`
  - "7日平均日增"卡片显示近 7 天日均新增用户数，如 `96`
  - "7日平均日增"卡片辅助指标显示周环比增长率，如 `+8.3%`
  - 三个卡片的数值与 Mock 数据接口返回值一致
- **验证方式**: 截图 + 文本断言
